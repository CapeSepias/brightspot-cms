Brightspot CMS is built on top of the Dari Framework. This install will create a new Brightspot CMS application and Maven project, as well as setting up Solr and H2 databases. It will all run within an embedded Tomcat application server using the Cargo plugin

The installation of Brightspot CMS requires two steps.

Step 1. Create your Project

Create your Maven project by running the archetype below from your command line. Replace yourGroupID with your own. Example - com.psddev and create your own project name. Note, the GroupID and Project name must not contain spaces or hyphens. If you are not familiar with Maven you can get an overview of the project layout structure here.

{% highlight java %}mvn archetype:generate -B \ -DarchetypeRepository=http://public.psddev.com/maven \ -DarchetypeGroupId=com.psddev \ -DarchetypeArtifactId=cms-app-archetype \ -DarchetypeVersion=2.1-SNAPSHOT \ -DgroupId=yourGroupID \ -DartifactId=yourProject{% endhighlight %}

Step 2. Run Project

Once you’ve created your project navigate to yourProject directory. Note, this directory will contain your pom.xml file.

Run mvn package cargo:run. This command will embed Brightspot CMS within your project and start the app with Tomcat on port 8080 - http://localhost:8080/cms

Note, Make sure you have nothing running on port 8080 before running this command.

Navigating to http://localhost:8080/cms will bring you to the default log in screen.

Log in
Follow the instructions to create an administrator account.

Sign in to finish the install. You are now ready to start creating your first project with Brightspot CMS. Head over to the Getting Started documentation to learn more about the CMS tool, or take a look at how to create your first object .

Starting and Stopping

To stop your project cancel the command line Ctrl-C. To restart the project, navigate to within yourProject directory and run mvn package cargo:run again.
